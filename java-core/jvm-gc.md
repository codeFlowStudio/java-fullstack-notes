# JVM 垃圾回收

垃圾回收（GC）是 JVM 自动管理内存的机制。

## GC 算法

### 标记-清除算法
标记存活对象，清除未标记对象。

### 标记-复制算法
将内存分为两块，存活的复制到另一块。

### 标记-整理算法
标记后，将存活对象向一端移动。

## 垃圾收集器

- **Serial**：单线程
- **Parallel**：多线程并行
- **CMS**：低延迟
- **G1**：分区收集，平衡吞吐量和延迟
- **ZGC**：超低延迟（JDK 11+）

## GC 调优参数

```bash
# 使用 G1 收集器
-XX:+UseG1GC

# 设置最大停顿时间
-XX:MaxGCPauseMillis=200
```
